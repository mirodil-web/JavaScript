JS oid umumiy ma'lumotlar

JavaScript, brauzerda yoki brauzerdan tashqari muhitlarda ishlaydigan skript tili sifatida ishlatiladi. U to'g'ridan-to'g'ri HTML va CSS bilan bog'liq veb sahifalarni interaktiv qilishga imkon beradi. JavaScript, Netscape Communications Corporation tomonidan 1995-yilda ishlab chiqilgan. Hozirda esa, brauzerlarda ommalashtirilgan va iste'molchilar tomonidan ko'p qo'llaniladi. JavaScript, obyektoriyentatsiyalangan dasturlash tilidir. Uning sintaksisi, C va Java kabi dasturlash tillari bilan keng o'xshashliklarga ega. JavaScript, HTML sahifalarda amalga oshirilgan har qanday amallarni (ko'rinish o'zgartirish, matnlarni tahrirlash, formalar bilan ishlash va h.k.) dasturlash imkoniyatini beradi. Uchta asosiy JavaScript kutubxonasi mavjud: ECMAScript, Document Object Model (DOM) va Browser Object Model (BOM). ECMAScript, JavaScriptning standart versiyasidir. DOM, HTML sahifalarini strukturaviy qismi bilan ishlashda yordam beradi. BOM esa brauzer funktsiyalariga murojaat qilish imkonini beradi. JavaScript, veb ilovalar yaratishning yanada kuchli yo'nalishi bo'lib, hozirgi vaqtda endi server tomonida ham (Node.js) ishlatiladi. JavaScript yaratilganidan buyon ko'p kutubxonalar, frameworklar va bibliotekalar mavjud bo'lib, ulardan ba'zilari: React, Angular, Vue.js, Express.js, jQuery, Bootstrap va ko'p boshqalaridir.
